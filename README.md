Kami menggunakan *Advanced Encryption Standard* (AES) dengan panjang kunci 256-bit. Sistem ini bekerja secara asimetris pada tingkat pengamanan tertinggi dan diakui secara global sebagai perlindungan data *Quantum-Resistant* (tahan terhadap serangan *brute-force* komputasi tingkat lanjut).

### Mekanisme Pengacakan Data (14 Putaran Siklus)
Data mentah Anda tidak sekadar disembunyikan, melainkan dihancurkan secara logis dan disusun ulang melalui operasi matriks matematika yang kompleks:
1. **SubBytes:** Substitusi byte non-linear untuk memutus pola data.
2. **ShiftRows:** Pergeseran baris data secara terstruktur untuk mengacak posisi karakter.
3. **MixColumns:** Penggabungan kolom data melalui operasi aljabar untuk menyebarkan bit.
4. **AddRoundKey:** Penguncian akhir menggunakan *Secret Key* unik yang hanya dimiliki oleh klien.

* **Zero-Knowledge Architecture:** Kami tidak menyimpan kunci dekripsi Anda di server kami. Anda memegang kendali penuh atas *Master Secret Key*.
* **SHA-512 Checksum:** Setiap paket data yang dienkripsi akan divalidasi dengan *hash* SHA-512 untuk memastikan tidak ada satu bit pun yang termodifikasi atau rusak selama proses penyimpanan.
* **End-to-End Secure:** Perlindungan mutlak untuk data saat diam (*Data at Rest*) maupun saat berpindah (*Data in Transit*).
